let, var and const

let const var
Can be declared without Initialization. Ex: var a; Can be declared without Initialization Ex: let a; Can not be declared without Initialization. Declaration and Initialization must be happen at a time. Ex: const a = 10
Default value of variable var a is undefined . Default value of variable let a is undefined. Default value of const a is Assigned value.
Accessing var a variable without Declaration gives output as undefined (Hoisting) . Accessing let a variable without Declaration gives an error called variable is not defined.
